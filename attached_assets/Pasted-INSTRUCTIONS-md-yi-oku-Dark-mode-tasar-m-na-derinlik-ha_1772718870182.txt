=== 4. ANİMASYONLAR ===

Sayfa geçişlerinde fade-in:
- client/src/components/layout/app-layout.tsx'de main content'e:
  className="animate-in fade-in duration-200"
  
  Veya basit CSS:
  @keyframes fadeIn {
    from { opacity: 0; transform: translateY(4px); }
    to { opacity: 1; transform: translateY(0); }
  }
  .page-fade-in { animation: fadeIn 0.2s ease-out; }

Stat kartları yüklenirken:
- Stagger animasyonu: her kart 50ms arayla belirsin
- className="animate-in fade-in duration-300" style={{ animationDelay: `${index * 50}ms` }}

Grafik kartları:
- Grafikler yüklenirken skeleton → chart geçişinde fade

Badge pulse (dikkat çekmesi gerekenler):
- Expiring soon: animate-pulse (yavaş)
- Overdue/expired: ring-2 ring-red-500/50 animate-pulse

=== 12. IMPORTANT ===

- Bu değişiklikler GLOBAL — tüm sayfaları etkiler
- shadcn/ui Card component'ini customize ediyorsan, variants ekle (default, hover, glow)
- Tailwind transition-all duration-200 her hover efektine ekle (smooth geçiş)
- Animasyonlar hafif olsun — 150-300ms arası, ease-out
- Performans: transform ve opacity animasyonları GPU-accelerated, box-shadow dikkatli kullan
- Mobile'da hover efektleri çalışmaz — focus state'leri de ayarla
- Mevcut fonksiyonelliği BOZMA — sadece CSS/class değişiklikleri
- Aşırıya kaçma — subtle > flashy
