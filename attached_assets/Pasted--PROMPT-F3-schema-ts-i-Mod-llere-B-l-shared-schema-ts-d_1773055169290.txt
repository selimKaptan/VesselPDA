## ⭐ PROMPT F3 — schema.ts'i Modüllere Böl

Her modül dosyasında gerekli cross-referanslar için diğer modüllerden import yapılacak. Örneğin proforma.ts'de vessels ve ports'a referans var, o yüzden:
import { vessels } from "./vessel";
import { ports } from "./port";
import { users, companyProfiles } from "../models/auth";

ÖNEMLİ: Mevcut shared/schema.ts'i sil ve yerine shared/schema/index.ts koy. Projede "@shared/schema" import path'i aynı kalacak çünkü tsconfig'de paths ayarı var.

ÖNEMLİ: Circular dependency'lere dikkat et. Eğer A → B ve B → A referans varsa, ikisini aynı dosyaya koy.
