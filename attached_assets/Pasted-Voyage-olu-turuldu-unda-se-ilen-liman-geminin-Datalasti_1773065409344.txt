DÜZELTME 1 — Backend: server/routes/tracking.routes.ts dosyasında "/api/vessels/live-position" endpoint'ini bul. Response JSON'ına voyage bilgisini de ekle:

Mevcut response'un sonuna şu alanları ekle (res.json bloğunun içine):

// Voyage bilgisi — kullanıcının oluşturduğu sefer
let activeVoyage = null;
try {
  const { voyages, ports } = await import("@shared/schema");
  const { eq, and, or, desc } = await import("drizzle-orm");
  const { db } = await import("../db");
  
  // Bu IMO'ya sahip vessel'ın aktif voyage'ını bul
  const vesselRecord = await db.select().from(
    (await import("@shared/schema")).vessels
  ).where(
    eq((await import("@shared/schema")).vessels.imoNumber, imo)
  ).limit(1);
  
  if (vesselRecord.length > 0) {
    const [voy] = await db.select().from(voyages).where(
      and(
        eq(voyages.vesselId, vesselRecord[0].id),
        or(
          eq(voyages.status, "planned"),
          eq(voyages.status, "in_progress"),
          eq(voyages.status, "active")
        )
      )
    ).orderBy(desc(voyages.createdAt)).limit(1);
    
    if (voy) {
      let portName = null;
      if (voy.portId) {
        const [p] = await db.select({ name: ports.name }).from(ports).where(eq(ports.id, voy.portId));
        portName = p?.name || null;
      }
      activeVoyage = {
        id: voy.id,
        portName: portName,
        eta: voy.eta,
        etd: voy.etd,
        status: voy.status,
        purposeOfCall: voy.purposeOfCall,
        cargoType: voy.cargoType,
      };
    }
  }
} catch (err) {
  // Voyage bilgisi opsiyonel, hata olursa sessizce devam et
}

Sonra res.json() bloğuna şu alanları ekle:

Destination bölümünde şu pattern'ı uygula:

{/* AIS Destination — geminin gerçek hedefi */}
<div>
  <span className="text-muted-foreground text-xs">AIS Destination</span>

    <div className="flex gap-2 text-xs text-muted-foreground">
