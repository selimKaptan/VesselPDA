=== 8. GLOBAL CSS VARIABLES ===

Tutarlılık için CSS variable'lar tanımla — client/src/index.css veya globals.css:

:root {
  --glass-bg: rgba(15, 23, 42, 0.85);
  --glass-blur: 12px;
  --glass-border: rgba(51, 65, 85, 0.6);
  --glass-shadow: rgba(0, 0, 0, 0.6);
  --glass-hover: rgba(30, 41, 59, 0.6);
}

.glass-panel {
  background: var(--glass-bg);
  backdrop-filter: blur(var(--glass-blur));
  -webkit-backdrop-filter: blur(var(--glass-blur));
  border: 1px solid var(--glass-border);
  box-shadow: 0 25px 50px -12px var(--glass-shadow);
}

.glass-item:hover {
  background: var(--glass-hover);
  border-radius: 0.375rem;
}

Bu class'ları component'lerde kullanabilirsin: className="glass-panel"

=== 9. IMPORTANT ===

- shadcn/ui component'leri client/src/components/ui/ altında — doğrudan düzenle
- Her component'in mevcut variant'larını koru (default, destructive, outline vb.)
- backdrop-blur-md: Safari desteği için -webkit-backdrop-filter de ekle (CSS variable'da zaten var)
- Performans: backdrop-blur GPU-intensive — çok fazla aynı anda açık element olmasın
- Light mode varsa (theme toggle): light mode'da glassmorphism farklı olabilir — şimdilik sadece dark mode
- Tutarlılık: TÜM floating panel'ler aynı glass stili kullanmalı
- Mevcut fonksiyonelliği BOZMA — sadece style class'ları değiştir
- Test: bir dialog aç, arkasındaki içerik hafif bulanık görünmeli
- Mobile'da backdrop-blur bazı eski cihazlarda yavaş olabilir — sorun olursa mobile'da sadece solid bg kullan
