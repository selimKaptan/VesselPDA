KONTROL LİSTESİ:

1. SERVER BAŞLATMA:
   - npm run dev çalışıyor mu?
   - Eğer hata varsa düzelt
   - TypeScript hataları var mı? (tsc --noEmit ile kontrol et)
   - Eksik import'lar var mı?
   - Tanımsız değişken/fonksiyon kullanılıyor mu?

2. DATABASE:
   - shared/schema.ts dosyası hatasız mı?
   - Tüm tablo tanımları doğru mu?
   - Foreign key referansları doğru tablolara mı bakıyor?
   - drizzle-kit push çalışıyor mu? Hata varsa düzelt.
   - Yeni eklenen tablolar veritabanında oluşmuş mu?
   - Role enum değerleri: "ship_agent", "shipowner", "ship_broker", "ship_provider", "admin" doğru mu?

3. ROUTES — Her route dosyasını kontrol et:
   server/routes/ klasöründeki TÜM dosyalar:
   - Import'lar doğru mu? (storage fonksiyonları, middleware'ler)
   - requireRole middleware import edilmiş mi ve doğru kullanılıyor mu?
   - isAuthenticated middleware her korumalı endpoint'te var mı?
   - Response format tutarlı mı? (res.json kullanılıyor mu)
   - Error handling var mı? (try-catch)
   - Endpoint path'leri çakışıyor mu?

4. STORAGE — Her storage dosyasını kontrol et:
   server/storage/ klasöründeki TÜM dosyalar:
   - Drizzle query'leri doğru mu?
   - Tablo referansları schema ile eşleşiyor mu?
   - Return tipleri doğru mu?
   - Undefined dönebilecek yerler handle edilmiş mi?

5. FRONTEND SAYFALAR — Her sayfa dosyasını kontrol et:
   client/src/pages/ klasöründeki TÜM dosyalar:
   - Import'lar doğru mu? (component'ler, hook'lar)
   - useQuery/useMutation hook'ları doğru endpoint'e mi bakıyor?
   - API response format'ı beklenen şekilde mi parse ediliyor?
   - Loading state gösteriliyor mu?
   - Error state gösteriliyor mu?
   - Boş veri durumu handle ediliyor mu? (empty state)

6. FRONTEND COMPONENT'LER:
   client/src/components/ klasöründeki TÜM dosyalar:
   - Prop tipleri doğru mu?
   - Eksik prop geçilmiş mi?
   - undefined/null güvenli mi? (optional chaining kullanılıyor mu)
   - Event handler'lar doğru bağlanmış mı?

7. APP.TSX — Route tanımları:
   - Tüm sayfalar için route tanımlı mı?
   - Route path'leri sayfa component'leriyle eşleşiyor mu?
   - ProtectedRoute kullanılıyorsa doğru çalışıyor mu?
   - Lazy loading varsa doğru import ediliyor mu?

8. LAYOUT:
   - app-layout.tsx (veya app-sidebar.tsx) hatasız mı?
   - Top bar render ediliyor mu?
   - Module tabs render ediliyor mu?
   - Side panel render ediliyor mu?
   - Rol bazlı menü filtrelemesi çalışıyor mu?
   - Tüm menü linkleri doğru route'a gidiyor mu?

9. AUTH:
   - Login çalışıyor mu?
   - Register çalışıyor mu? 4 rol kartı görünüyor mu?
   - Logout çalışıyor mu?
   - Session korunuyor mu? (sayfa yenileyince login kalıyor mu)
   - Yetkisiz erişim engelliyor mu?

10. SOCKET.IO:
    - Bağlantı kuruluyor mu?
    - Hata veriyorsa düzelt veya graceful fallback ekle
    - Socket olmadan da sayfa çalışmalı

11. CONSOLE HATALARI:
    - Tarayıcı console'unda hata var mı?
    - React warning'leri var mı? (key prop, useEffect dependency)
    - Network tab'da failed request var mı?

DÜZELTME KURALLARI:
- Sadece HATA düzelt, yeni özellik EKLEME
- Eğer bir sayfa tamamen bozuksa, en azından hata vermeden yüklensin
- Eğer bir endpoint çalışmıyorsa, en azından 500 yerine anlamlı hata dönsün
- Eğer bir import eksikse, doğru dosyayı import et
- Eğer bir tablo eksikse, schema.ts'e ekle
- Eğer bir component bulunamıyorsa, basit placeholder oluştur
- TypeScript any kullanmak geçici olarak kabul edilebilir (sonra düzeltiriz)
