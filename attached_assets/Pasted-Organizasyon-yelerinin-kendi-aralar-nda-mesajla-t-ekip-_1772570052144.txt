Organizasyon üyelerinin kendi aralarında mesajlaştığı ekip chat sistemi ekle.

Yapılacaklar:

1. shared/schema.ts dosyasına şu tabloları ekle:

teamChannels tablosu:
- id (serial PK)
- organizationId (integer, FK organizations, not null)
- name (text, not null) — "genel", "operasyon", "muhasebe" vs.
- description (text, nullable)
- channelType (text, default "public") — "public" (tüm üyeler), "private" (sadece eklenenler)
- createdByUserId (varchar, FK users)
- createdAt (timestamp, default now)

teamChannelMembers tablosu (private kanallar için):
- channelId (integer, FK teamChannels)
- userId (varchar, FK users)
- joinedAt (timestamp, default now)
- PRIMARY KEY (channelId, userId)

teamMessages tablosu:
- id (serial PK)
- channelId (integer, FK teamChannels, not null)
- senderId (varchar, FK users, not null)
- content (text, not null)
- messageType (text, default "text") — "text", "file", "system"
- fileUrl (text, nullable)
- fileName (text, nullable)
- isEdited (boolean, default false)
- replyToId (integer, FK teamMessages, nullable) — mesaja yanıt
- createdAt (timestamp, default now)

2. Organizasyon oluşturulduğunda otomatik "Genel" kanalı oluştur.

3. Endpoint'ler (server/routes/team-chat.ts):

GET /api/organizations/:orgId/channels — Kanal listesi
POST /api/organizations/:orgId/channels — Yeni kanal oluştur
PATCH /api/organizations/:orgId/channels/:channelId — Kanal düzenle
DELETE /api/organizations/:orgId/channels/:channelId — Kanal sil

GET /api/organizations/:orgId/channels/:channelId/messages?page=1&limit=50 — Mesajlar (sayfalanmış, en yeniler son)
POST /api/organizations/:orgId/channels/:channelId/messages — Mesaj gönder
PATCH /api/team-messages/:id — Mesaj düzenle
DELETE /api/team-messages/:id — Mesaj sil (sadece kendi mesajını)

POST /api/organizations/:orgId/channels/:channelId/members — Private kanala üye ekle
DELETE /api/organizations/:orgId/channels/:channelId/members/:userId — Üye çıkar

4. Socket.io entegrasyonu (server/socket.ts güncelle):
   - Organizasyon üyeleri org-{orgId} room'una katılsın
   - Kanal mesajları channel-{channelId} room'una emit edilsin
   - Event'ler: "team_message", "team_typing", "channel_created"
   - Yazıyor göstergesi: 3 saniye timeout ile

5. client/src/pages/team-chat.tsx sayfası oluştur:
   - Sol panel: Kanal listesi (public ve private kanallar ayrı grup)
   - Sağ panel: Mesaj akışı
   - Mesaj baloncukları: gönderen adı, avatar, zaman, içerik
   - Mesaj gönderme alanı: input + gönder butonu + dosya ekleme
   - Mesaja yanıt verme (reply)
   - "Yazıyor..." göstergesi
   - Yeni kanal oluşturma dialog'u
   - Kanal üyeleri listesi (private kanallar için)

6. App.tsx'e /team-chat route'u ekle.
7. Sidebar'a "Ekip Chat" menü öğesi ekle (MessageCircle icon).
   Okunmamış mesaj sayısı badge göster.

Socket.io ile gerçek zamanlı mesajlaşma. Sayfa yenilemeye gerek kalmadan mesajlar gelsin.
