## 1. BULK CREW ENDPOINT EKSİK

server/routes/husbandry.routes.ts dosyasında /crew/bulk endpoint'i yok. Ekle:

Dosyada mevcut route tanımlarının arasına (tercihen en sona, export default router'dan ÖNCE) ekle:

// POST /:voyageId/crew/bulk — Toplu crew ekleme (AI paste)
router.post("/:voyageId/crew/bulk", async (req: any, res) => {
  try {
    const voyageId = parseInt(req.params.voyageId);
    const { crewMembers, signerType } = req.body;
    
    if (!Array.isArray(crewMembers) || crewMembers.length === 0) {
      return res.status(400).json({ error: "No crew members" });
    }
    
    const typeMap: Record<string, string> = {
      on_signer: "sign_on", off_signer: "sign_off",
      sign_on: "sign_on", sign_off: "sign_off",
    };
    
    const added = [];
    for (const m of crewMembers) {
      const type = typeMap[m.signerType || signerType] || "sign_on";
      try {
        const insertData: any = {
          voyageId,
          type,
          fullName: `${m.firstName || ""} ${m.lastName || ""}`.trim() || "Unknown",
          rank: m.rank || null,
          nationality: m.nationality || null,
          passportNo: m.passportNo || null,
          seamansBookNo: m.seamansBookNo || null,
        };
        if (m.passportExpiry) try { insertData.passportExpiry = new Date(m.passportExpiry); } catch {}
        if (m.dob) try { insertData.dob = new Date(m.dob); } catch {}
        if (m.birthPlace) insertData.birthPlace = m.birthPlace;
        if (m.employeeNo) insertData.employeeNo = m.employeeNo;
        if (m.flights?.[0]?.flightNo) insertData.flightCode = m.flights[0].flightNo;
        if (m.flights?.length > 0) insertData.flightDetails = JSON.stringify(m.flights);
        
        const [row] = await db.insert(crewChanges).values(insertData).returning();
        added.push(row);
      } catch (err: any) {
        console.error(`[crew-bulk] Failed ${m.lastName}:`, err.message);
      }
    }
    res.status(201).json({ added: added.length, crew: added });
  } catch (e: any) {
    res.status(500).json({ error: e.message });
  }
});

ÖNEMLI: crewChanges tablosunun import'unu dosyanın başında kontrol et. Yoksa ekle:

## 2. CARGO IMPORT HANDLER EKSİK

client/src/pages/voyage-detail.tsx dosyasında declaration modal var ama import handler fonksiyonu yok.

Dosyada "parsedParcels" state'ini bul. Aynı bölgeye handleImportParcels fonksiyonunu ekle:

async function handleImportParcels() {
  try {
    let imported = 0;
    for (const [i, parcel] of parsedParcels.entries()) {
      const res = await fetch(`/api/v1/voyages/${voyageId}/cargo-parcels`, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          receiverName: parcel.receiverName || "Unknown",
          shipperName: parcel.shipperName || null,
          cargoType: parcel.cargoType || "Unknown",
          cargoGrade: parcel.cargoGrade || null,
          targetQuantity: parcel.quantity || 0,
          blQuantity: parcel.quantity || 0,
          unit: parcel.unit || "MT",
          blNumber: parcel.blNumber || null,
          holdNumbers: parcel.holdNumbers || null,
          tankNumbers: parcel.tankNumbers || null,
          dischargeSequence: i + 1,
          loadingSequence: i + 1,
          notes: [
            parcel.hsCode ? `HS: ${parcel.hsCode}` : "",
            parcel.countryOfOrigin ? `Origin: ${parcel.countryOfOrigin}` : "",
          ].filter(Boolean).join(" · ") || null,
        }),
      });
      if (res.ok) imported++;
    }
    toast({ title: "Cargo Imported", description: `${imported} parcels created from declaration` });
    setShowDeclarationModal(false);
    setParsedParcels([]);
    queryClient.invalidateQueries();
  } catch (e: any) {
    toast({ title: "Error", description: String(e), variant: "destructive" });
  }
}

Sonra declaration modal'daki "Import" butonunu bul — onClick'ine bu fonksiyonu bağla:
onClick={handleImportParcels}

## 4. toFixed NULL KORUMASI (41 adet)

Tüm sayfalarda .toFixed() çağrılarını güvenli hale getir. Proaktif yaklaşım — helper fonksiyonu oluştur:

client/src/lib/safe-number.ts dosyası oluştur:

export function safeFixed(val: any, digits: number = 1): string {
  const num = Number(val);
  return isNaN(num) ? "0" : num.toFixed(digits);
}

export function safeLocale(val: any): string {
  const num = Number(val);
  return isNaN(num) ? "0" : num.toLocaleString();
}

Sonra voyage-detail.tsx, pms.tsx, passage-planning.tsx, cargo-operations.tsx, voyage-pnl.tsx dosyalarında:

Tehlikeli kullanımları bul ve düzelt:
ÖNCE: value.toFixed(1)  
SONRA: (value ?? 0).toFixed(1)

ÖNCE: amount.toLocaleString()
SONRA: (amount ?? 0).toLocaleString()

Veya toplu düzeltme: Tüm .toFixed( kullanımlarının önüne null check ekle:
grep -rn "\.toFixed(" client/src/pages/*.tsx komutuyla bul ve her birinde değişkenin önüne (variable ?? 0) veya Number(variable || 0) ekle.

En kolay yol — her dosyada replace:
.toFixed( → öncesindeki değişkene ?? 0 ekle
Örnek: progress.toFixed(1) → (progress ?? 0).toFixed(1)
Örnek: rate.toFixed(2) → (rate ?? 0).toFixed(2)
