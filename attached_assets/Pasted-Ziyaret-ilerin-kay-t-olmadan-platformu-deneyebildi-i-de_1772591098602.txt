2. server/routes — demo endpoint'lerini güncelle:

   POST /api/demo/start — Demo session başlat
   - body: { role: "ship_agent" | "shipowner" | "ship_broker" | "ship_provider" }
   - Admin seçeneği YOK
   - Yeni demo session ID oluştur
   - Seçilen role göre demo kullanıcı ata
   - Demo token döndür (24 saat geçerli)
   - Response: { token, user: { name, role, organizationName } }

   POST /api/demo/switch-role — Demo'da rol değiştir
   - body: { role: "ship_agent" | "shipowner" | "ship_broker" | "ship_provider" }
   - Admin seçeneği YOK
   - Farklı perspektiften görsün
   - Aynı demo verileri, farklı dashboard ve menüler

   POST /api/demo/reset — Demo verilerini sıfırla
   
   GET /api/demo/status — Demo session durumu
