=== ANA LAYOUT ===

client/src/components/layout/app-layout.tsx yeniden yaz:

import { useState } from 'react';
import { TopBar } from './top-bar';
import { IconRail } from './icon-rail';
import { SidebarPanel } from './sidebar-panel';
import { useMobile } from '@/hooks/use-mobile';

export function AppLayout({ children }) {
  const [activeCategory, setActiveCategory] = useState('operations');
  const isMobile = useMobile();

  if (isMobile) {
    // Mobile: hamburger menü → Sheet ile full sidebar
    // Bottom nav: Dashboard, Operations, Messages, Fleet, More
    return (
      <div className="min-h-screen bg-[#0B1120]">
        <MobileTopBar onMenuClick={() => setSidebarOpen(true)} />
        <main className="pt-14 px-3 pb-20">{children}</main>
        <MobileBottomNav />
        <Sheet ...> full sidebar </Sheet>
      </div>
    );
  }

  return (
    <div className="min-h-screen bg-[#0B1120] flex flex-col">
      <TopBar />
      <div className="flex flex-1">
        <IconRail activeCategory={activeCategory} onCategoryChange={setActiveCategory} />
        <SidebarPanel activeCategory={activeCategory} />
        <main className="flex-1 p-6 overflow-y-auto">
          {/* Breadcrumb */}
          <div className="flex items-center gap-2 text-xs text-slate-500 mb-4">
            <span>{getCategoryLabel(activeCategory)}</span>
            <span>/</span>
            <span className="text-white">{getCurrentPageLabel()}</span>
          </div>
          {children}
        </main>
      </div>
    </div>
  );
}

=== BREADCRUMB ===

Her sayfanın üstünde: Operations / Proformas (PDA) gibi breadcrumb
- Kategori adı (tıklanabilir, slate renk)

- Sayfa adı (beyaz, aktif)

=== BADGE VERİLERİ ===

Sidebar badge'leri gerçek veriden gelsin:
- Messages badge: GET /api/messages/unread-count → unread sayısı
- Tenders badge: aktif tender sayısı (veya bağlam)
- Certificates badge: GET /api/certificates/expiring?days=30 → "!" uyarısı
- Nominations badge: GET /api/nominations/pending-count

Bu verileri useQuery ile çek, sidebar'da göster.
Veri yoksa veya 0 ise badge gösterme.
