=== EMPTY STATE COMPONENT ===

client/src/components/empty-state.tsx oluştur:

interface EmptyStateProps {
  icon: string;
  title: string;
  description: string;
  actionLabel?: string;
  actionHref?: string;
  onAction?: () => void;
  secondaryLabel?: string;
  secondaryHref?: string;
  tips?: string[];
}

function EmptyState({ icon, title, description, actionLabel, actionHref, onAction, secondaryLabel, secondaryHref, tips }: EmptyStateProps) {
  return (
    <div className="flex flex-col items-center justify-center py-16 px-4">
      <div className="text-6xl mb-4">{icon}</div>
      <h3 className="text-lg font-semibold text-white mb-2">{title}</h3>
      <p className="text-sm text-slate-400 text-center max-w-md mb-6">{description}</p>
      
      <div className="flex gap-3">
        {actionLabel && (
          actionHref ? (
            <Link href={actionHref}>
              <Button className="bg-sky-500 hover:bg-sky-600">{actionLabel}</Button>
            </Link>
          ) : (
            <Button onClick={onAction} className="bg-sky-500 hover:bg-sky-600">{actionLabel}</Button>
          )
        )}
        {secondaryLabel && secondaryHref && (
          <Link href={secondaryHref}>
            <Button variant="outline">{secondaryLabel}</Button>
          </Link>
        )}
      </div>

      {tips && tips.length > 0 && (
        <div className="mt-8 bg-slate-800/30 border border-slate-700/30 rounded-xl p-4 max-w-md">
          <p className="text-xs font-semibold text-slate-300 mb-2">💡 Quick Tips</p>
          <ul className="space-y-1">
            {tips.map((tip, i) => (
              <li key={i} className="text-xs text-slate-400 flex items-start gap-2">
                <span className="text-sky-400 mt-0.5">•</span>
                <span>{tip}</span>
              </li>
            ))}
          </ul>
        </div>
      )}
    </div>
  );
}

=== IMPORTANT ===

- EmptyState component'i reusable olsun — her sayfada import et
- Mevcut data varsa EmptyState gösterme, normal liste göster
- Kontrol: items && items.length === 0 ise EmptyState
- Loading durumunda EmptyState değil Skeleton göster
- Tips bölümü opsiyonel — bazı sayfalarda olmayabilir
- Mevcut create dialog/modal'ları koru — onAction ile tetikle
- actionHref Link ile, onAction fonksiyon ile çalışsın
- Responsive: mobile'da da güzel görünsün
- Dark tema uyumlu renkler
